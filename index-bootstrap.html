This is Wordlist Wrangler. The application allows to extract N-Grams, detect letters with a RegEx and transliterate with a custom orthography profile. To start, upload a .txt file with the data. The orthography profile can be created manually or uploaded from your computer, please use .tsv or .csv with tabs(" "), commas(",") or semicolons(";") as delimiters. Press Download to download the output.
